# Variables del Dataset

## Variables Numéricas

* `Edad` - Edad del cliente
* `Ingreso_Anual` - Ingresos anuales
* `Total_Gastado` - Total de dinero gastado
* `Numeros_Compras` - Cantidad de compras
* `Importe_Promedio_Transaccion` - Promedio por compra
* `Numero_de_Devoluciones` - Cantidad de devoluciones
* `Cantidad_de_Contactos_Soporte` - Contactos con soporte
* `Anios_como_cliente` - Años de antigüedad
* `Ultimo_dia_compra` - Días desde última compra
* `Nivel_Satisfaccion` - Nivel de satisfacción

## Variables Categóricas

* `Genero` - Género del cliente
* `Esta_Suscripto_via_Email` - Suscripción a emails (Sí/No)
* `Respuesta_a_Promocion` - Respuesta a promociones (Sí/No)
* `Target` - Variable objetivo a predecir (Churn)
